# 🌐 Networking Protocols & Ports

## 📊 Common Protocols

| Protocol | Transport | Port |
| -------- | --------- | ---- |
| TELNET   | TCP       | 23   |
| DNS      | TCP/UDP   | 53   |
| HTTP     | TCP       | 80   |
| HTTPS    | TCP       | 443  |
| FTP      | TCP       | 21   |
| SMTP     | TCP       | 25   |
| POP3     | TCP       | 110  |
| IMAP     | TCP       | 143  |

---

## 🔒 Secure Versions

| Protocol | Port      |
| -------- | --------- |
| HTTPS    | 443       |
| SMTPS    | 465 / 587 |
| POP3S    | 995       |
| IMAPS    | 993       |

---

## 💡 Notes

* HTTP is not secure (no encryption)
* HTTPS uses TLS encryption
* DNS resolves domain names to IP addresses
